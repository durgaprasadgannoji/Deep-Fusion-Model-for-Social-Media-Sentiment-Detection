# Deep Fusion Social Media Sentiment Detection

## Overview

This project presents a multimodal sentiment analysis framework that combines textual and visual information from social media content to improve sentiment classification accuracy.

The model integrates RoBERTa-based Natural Language Processing and ResNet-based Computer Vision features through an attention-based deep fusion architecture to classify sentiments as Positive, Negative, or Neutral.

---

## Problem Statement

Traditional sentiment analysis relies only on textual information and often fails to capture the complete context of social media posts containing images and text.

This project addresses the challenge by leveraging multimodal deep learning techniques.
